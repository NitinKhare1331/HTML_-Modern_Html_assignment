Various features of HTML 5 are :-

  1. It improved multimedia support using media tags like
  2. It added new structural elements such as header , nav , article , footer , etc.
  3. It introduced progress tag which is used to check the progress of the task while execution.
  4. It introduced email type for input tag.
  5. Geo location API was introduced so that user can access to location based services.
  6. It improved accessibility.

3 Key Features of HTML 5 are:-

  1. Nav Tag

    The nav tag helps in defining a set of navigation links. It helps create a website section containing navigation links (within the current document or to another document). A few examples of navigation links are the table of contents and indexes.

  2. Header

    The header element in HTML5 contains introductory content. It can include a set of introductory or navigational support on a website. Typically, the header element contains heading elements, logos or icons, a search form, author's information, etc.

  3. Footer

    The footer tag defines the footer of a document or a section. Typically, the footer contains information related to the author, copyright, contact information, sitemap, related documents, back-to-top links, etc. There can be more than one footer element in one document.
